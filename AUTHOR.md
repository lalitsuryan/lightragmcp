## Lalit Suryan

**Creator and Lead Developer** of LightRAG MCP Server

### About

Lalit Suryan is the creator and maintainer of the LightRAG MCP Server, a comprehensive Model Context Protocol implementation for LightRAG integration. This project provides 30+ tools for document management, knowledge graph operations, and RAG capabilities through a clean MCP interface.
